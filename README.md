# Nonware

Monorepo for the Nonware portfolio platform.

## Apps

- `backend/` — Django backend APIs and business logic
- `web/` — Next.js frontend
- `mobile/` — Flutter mobile app
- `infra/` — deployment and CI/CD related files

## Stack

- Django
- Next.js
- Flutter
- Docker
- PostgreSQL
- Google Cloud Run
- GitHub Actions
