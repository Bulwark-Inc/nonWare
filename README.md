# Nonware

Monorepo for the Nonware portfolio platform.

## Structure

Nonware contains three applications.

backend/ — Django backend APIs  
web/ — Next.js frontend website  
mobile/ — Flutter mobile application  

## Tech Stack

Backend
- Django
- PostgreSQL
- Docker

Frontend
- Next.js
- TailwindCSS

Infrastructure
- Docker
- GitHub Actions
- Google Cloud Run
